# Lumina Audio - Optimeret og Redesignet Version  
Dette eksamensprojekt er en optimeret og gennemarbejdet version af min gruppes tidligere landingpage-projekt. Formålet med optimeringen har været at forbedre projektets struktur, kodekvalitet og visuelle identitet samt at skabe en mere professionel og skalerbar løsning.

Den nye version kombinerer et fuldt redesign udviklet i Figma med en systematisk omskrivning af HTML-, CSS- og tilføjelse af JavaScript.

---

## 1. Formål med optimeringen  
Den oprindelige version af projektet indeholdt en rodet mappestruktur, begrænset brug af moderne CSS-teknikker og ingen JavaScript-funktionalitet. Koden var desuden ikke struktureret efter genanvendelighed eller udviklingsprincipper som modularitet og separation of concerns.

Målet med den optimerede version har derfor været at:

- Forbedre den overordnede kodekvalitet  
- Strukturere projektet logisk og skalerbart  
- Implementere et nyt og moderne visuelt design  
- Øge vedligeholdelsesvenligheden gennem konsistent navngivning og mappestruktur  
- Tilføje JavaScript-funktionalitet, som ikke fandtes i den gamle version  
- Udnytte CSS-variabler og moderne stylingprincipper  
- Sikre bedre performance og optimerede assets
